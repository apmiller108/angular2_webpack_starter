Hello, this is Angular 2 with Webpack

What is this?

This boilerplate follows the angular.io webpack introduction with some minor changes, as well as the inclusion of sass loading and other stuff (see below). See the docs if you want.

What's included:

  • Typescript
  • An opinionated Tslint configuration
  • Karma for running tests
  • Jasmine testing framework
  • Webpack configurations for test, development and production environments
  • Flexbox Grid
